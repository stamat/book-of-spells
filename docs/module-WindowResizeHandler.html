    Array<string | QueryEntryObject>

    Example
    const query = this._parseMediaQueryString('(min-width: 768px) and (max-width: 1024px)')

    (private, inner) validateMediaQuery(query)

    Perform a query. You can use this to check if the window width or height is within a certain range by providing a query array. Query arrays are parsed from media query strings using _parseMediaQueryString and contain query entry objects QueryEntryObject and strings for the operators 'and' and 'or'.
